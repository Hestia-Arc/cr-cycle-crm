Cr-cycle Real Estate CRM (Frontend) built with Nextjs, Shadcn-UI.

UI (landing page) inspired by Khoa "JAK" [https://www.figma.com/@wblekhoa]

This application is a Sales Customer Relationship Management (CRM) tool designed to help users manage their sales processes, track deals, and maintain customer relationships. The app features a modern interface that allows users to navigate through various sections, including a dashboard, deals, contacts, tasks, and integrations.

To use the app, users can log in and access the main dashboard, which provides an overview of key metrics such as monthly revenue, new deals closed, and conversion rates. Users can add new deals, contacts, and tasks through intuitive forms and manage existing entries with options to edit or delete them. The app also includes filtering and searching capabilities to help users quickly find specific deals or contacts.

The application is built using React and TypeScript, leveraging Next.js for server-side rendering and routing. It utilizes various UI components from a custom design system, including cards, buttons, inputs, and modals, to create a cohesive user experience. The app also integrates with third-party services like Slack, Google Calendar, and Zapier to enhance productivity and streamline workflows.

Key features of the app include:

- A responsive dashboard displaying sales metrics and recent activities.
- A deals management section where users can view, add, edit, and delete deals, along with filtering options based on deal stages.
- A contacts management section for maintaining customer information, with similar functionalities for adding, editing, and deleting contacts.
- A tasks management section to track to-dos, with options to mark tasks as complete and filter by status.
- Integration capabilities with popular tools to enhance collaboration and productivity.
- A sidebar for easy navigation between different sections of the app.

Overall, this Sales CRM application aims to provide a comprehensive solution for sales teams to manage their workflows efficiently and effectively.
