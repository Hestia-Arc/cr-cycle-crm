Cr-cycle Real Estate CRM (Frontend) built with Nextjs, Shadcn-UI.

UI (landing page) inspired by Khoa "JAK" [https://www.figma.com/@wblekhoa]

This application is a Customer Relationship Management (CRM) tool designed to help real-estate agents manage their sales processes, track deals, and maintain customer relationships. The app features a modern interface that allows users to navigate through various sections, including a dashboard, deals, contacts, tasks, and integrations.
